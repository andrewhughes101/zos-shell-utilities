# zos-shell

A collection of z/OS shell utilities for mainframe system administration.

## Tools

- **[inuse](bin/inuse)** - Check which jobs/systems are using a z/OS dataset
- **[logr_manager](bin/logr_manager)** - Manage LOGR structures and log streams using IXCMIAPU
